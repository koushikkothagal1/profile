About Me

Koushik Kothagal is the founder of Java Brains, an online training website that offers courses on various enterprise Java and JavaScript technologies. He works as a Staff Engineer at LinkedIn, has over 20 years of professional experience working on full-stack web applications and has worked extensively with technologies such as Java, Spring, Java EE, JavaScript, Angular and React.

He loves teaching, and when he's not coding Java and JavaScript, he's probably teaching it! He currently lives in the San Francisco Bay Area.
